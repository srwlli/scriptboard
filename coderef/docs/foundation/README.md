# Scriptboard

> Modern workspace for managing LLM prompts, attachments, and responses. Desktop-first (Electron) with Python Core, FastAPI backend, and Next.js frontend.

## üöÄ Quick Start

### Prerequisites
- Python 3.10+
- Node.js 18+
- npm or yarn

### Installation

```bash
# Backend
cd backend
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
pip install -r requirements.txt

# Frontend
cd ../frontend
npm install

# Electron (optional for desktop app)
cd ../shell
npm install
```

### Usage

```bash
# Start Backend
cd backend
venv\Scripts\activate  # or source venv/bin/activate
uvicorn api:app --reload --host 127.0.0.1 --port 8000

# Start Frontend (in a second terminal)
cd frontend
npm run dev

# Start Electron (optional)
cd shell
npm run dev
```
